Se desarrolla una app para el modelo de negocios de un Jardin de Infantes Privado de la Provincia de Córdoba, Argentina.

Modelos (se indican con * al inicio, los atributos): 
- Salas: No se agrega formulario para agregar, porque considero que son las únicas 3 salas que pueden existir en la Institución.
	* Nombre
- Alumnos:
	* Nombre
	* Apellido
	* Edad
	* Tutor
- Maestras:
	* Nombre
	* Apellido
	* Edad
	* Email

- Eventos:
	* Nombre
	* Fecha

En la página de Inicio se podrán buscar alumnos por algún patrón de búsqueda que considere el usuario.
